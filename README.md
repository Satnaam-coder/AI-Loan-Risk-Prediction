# 💳 AI Loan Risk Prediction System

## 📌 Project Overview
This project is a Machine Learning based web application that predicts whether a loan applicant is risky or safe. It helps financial institutions make better decisions by analyzing user data.

---

## 🚀 Features
- 🔐 User Login & Signup System
- 👑 Admin Panel (Dashboard Access)
- 🤖 Machine Learning Prediction
- 📊 Risk Classification (Low / Medium / High)
- 📈 Dashboard with Analytics
- 💾 Data Storage using SQLite
- 📥 CSV Data Download (Admin Feature)

---

## 🧠 Machine Learning Model
- Algorithm Used: Random Forest
- Output:
  - 🟢 Low Risk
  - 🟡 Medium Risk
  - 🔴 High Risk
- Probability-based prediction for better accuracy

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS
- Backend: Python (Flask)
- Database: SQLite
- Machine Learning: Scikit-learn
- Visualization: Chart.js

---
## 👤 User Roles

### 👤 Normal User
- Login and predict loan risk

### 👑 Admin
- Access dashboard
- View all records
- Download data (CSV)
